# Security Policy

AetherFlow AI is committed to maintaining a secure and reliable platform for developer and enterprise workloads. This document details our security practices and reporting pathways.

---

## 🔒 Supported Versions

We actively maintain security updates for the following version tracks:

| Version | Supported |
|---|---|
| v1.0.x |  Active Support |
| < v1.0.0 |  End of Life (EOL) |

---

## 🤝 Reporting a Vulnerability

If you discover a security vulnerability in this project:
1.  **Do not open a public issue** on GitHub.
2.  Email your findings directly to our security team at **security@aetherflow.ai**.
3.  Include:
    *   A detailed description of the vulnerability.
    *   Steps to reproduce the vulnerability (proof of concept code or instructions).
    *   The potential impact of the issue.

We will acknowledge receipt of your report within 24 hours and provide an estimated fix timeline. We request that you follow responsible disclosure principles until a patch has been released.
